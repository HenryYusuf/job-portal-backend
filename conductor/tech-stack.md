# Tech Stack: Job Portal Backend

## 1. Core Technologies
- **Runtime:** **Bun** - A fast, all-in-one JavaScript/TypeScript runtime.
- **Backend Framework:** **Hono** - A lightweight, ultra-fast web framework with built-in support for Bun.
- **Language:** **TypeScript** - For robust typing and developer productivity.

## 2. Infrastructure & Persistence
- **Database:** **PostgreSQL** - A powerful, relational database for reliable storage.
- **ORM/Query Builder:** **Drizzle ORM** with **postgres.js** - To interact with PostgreSQL.

## 3. Identity & Access
- **Authentication:** **better-auth** - A modern authentication library supporting Email/Password and Google OAuth integration.

## 4. API & Documentation
- **Validation:** **Zod** - For schema-based validation of API requests and responses.
- **Documentation:** **@hono/zod-openapi** - For automated, type-safe OpenAPI documentation.
